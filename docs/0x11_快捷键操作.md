# 快捷键操作

## 常用快捷键：

1. `.`：跳转到`github.dev`页面。

2. `g` + `i`：跳转到 issues 页面。

3. `c`：创建 `issue`

除了以上常用快捷键，Github 还提供了一些其他快捷键，如查看文件历史记录、打开快捷键菜单等。您可以通过按下 `?` 来查看所有可用的快捷键。

## 快捷键菜单

按下 `?` 键 或者  `shift` + `/` 键 ，打开 Github 的快捷键菜单。

## 仓库代码查看

进入仓库后

按 【T】键 快速筛选文件

进入文件后 按 【L】键快速进入 某一行

点击行号 可以快速复制此行代码

按 【B】键可以快速查看该文件的改动记录
